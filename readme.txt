=== Show/Hide WordPress Admin Bar ===
Contributors: James Mountford
Tags: show, hide, admin bar.
Requires at least: 3.1
Tested up to: 3.1
Stable tag: 1.1

This plugin adds a little tab beneath the admin bar in WP 3.1 onwards which on click shows/hides the admin bar.

Apologies to those who downloaded it yesterday (01/03/2010). Just tested it out from the repository and realised it didnt work because the folder name that WordPress provided was different to the folder name that I was building it in so a link was incorrect.
The path has been corrected now.
